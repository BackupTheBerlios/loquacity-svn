File Permissions

Loquacity need to be able to write to disk to store it's cache of templates, and if you want to use the blo.gs favorites functionality.

We will now check the permissions of the cache folder, the compiled_templates folder, and the cache/favorites.xml file. They all need to be writable by the webserver.

Additionally, config.php should be writable during the install. At the end of the install when the config file is written to disk, you should change the permissions back so it is not writable by the webserver.
