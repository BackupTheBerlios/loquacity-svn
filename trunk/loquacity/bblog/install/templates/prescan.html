File Permissions

Loquacity requires the ability to write to disk in order to store its cache of templates or if you want to use the blo.gs favorites functionality.

We will now check the permissions of the generated folder and its contents to make sure the webserver can write to them.

Additionally, config.php should also be writable. At the end of the install when the config file is written to disk, you should change the permissions back so it is not writable by the webserver anymore.
